# Propuesta para Inversores - Asistente IA Geriátrico

## Resumen Ejecutivo

| Concepto | Descripción |
|:---|:---|
| **Proyecto** | Asistente IA Geriátrico - Dispositivo intra-auricular para adultos mayores |
| **Problema** | 55M personas con demencia en el mundo, 30% de mayores de 65 años se caen al año |
| **Solución** | Dispositivo sub-100 USD que lee etiquetas, monitorea glucosa, detecta caídas y habla con voz familiar |
| **Mercado** | 800M adultos mayores globalmente, 70M con diabetes, 55M con demencia |
| **Estado** | Diseño conceptual completo, patente provisional, documentación técnica |
| **Inversión requerida** | 400,000 USD |
| **ROI proyectado (5 años)** | 4,200% para sistema de salud, 78% TIR para inversores |

## El Problema

### Datos Globales
| Indicador | Valor | Fuente |
|:---|:---|:---|
| Adultos mayores (65+) | 800 millones | OMS |
| Con demencia | 55 millones | OMS |

| Adultos con diabetes | 70 millones | IDF |
| Mayores que se caen al año | 30% | CDC |
| Fracturas de cadera por caídas | 300,000/año (EE.UU.) | CDC |

### El Dolor Humano
- **Adulto mayor:** Pierde autonomía, se siente solo, tiene miedo
- **Familiar:** Agotamiento, culpa, no puede estar siempre
- **Sistema de salud:** Costos altos, camas ocupadas, personal insuficiente

---

## La Solución

### Asistente IA Geriátrico
Un dispositivo intra-auricular que:

| Función | Beneficio |
|:---|:---|
| **Visión artificial** | Lee etiquetas para quien no ve bien |
| **Monitoreo de glucosa** | Alerta hipoglucemias antes de que ocurran |
| **Detección de caídas** | Avisa inmediatamente si hay accidente |
| **Memoria afectiva** | Recuerda recetas, rutinas, nombres |
| **Voz familiar** | Habla con la voz del hijo, la hija, el nieto |
| **Sin internet** | 100% privado, funciona sin conexión |
| **Precio** | Menos de 100 USD (vs 400+ de alternativas) |
