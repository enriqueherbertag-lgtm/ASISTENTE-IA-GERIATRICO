## Ruta en Chile (ISP)

### Paso 1: Clasificación
- Determinación de clase: Clase IIa
- Basado en: dispositivo activo no invasivo, contacto con piel, duración limitada

### Paso 2: Sistema de Calidad
- Implementación de ISO 13485
- Certificación por entidad acreditada (6-9 meses)

### Paso 3: Expediente Técnico
| Documento | Contenido |
|:---|:---|
| Descripción del dispositivo | Funcionamiento, indicaciones |
| Especificaciones técnicas | Diseño, materiales, software |
| Verificación y validación | Pruebas de laboratorio |
| Gestión de riesgos | ISO 14971 |
| Biocompatibilidad | ISO 10993 |
| Estabilidad | Pruebas de envejecimiento |
| Rotulado e instructivos | Manuales de usuario |
| Evaluación clínica | Literatura + estudio propio |

### Paso 4: Evaluación ISP
- Revisión documental (90-180 días)
- Inspección de fábrica (si aplica)
- Resolución de aprobación

**Tiempo estimado total:** 12-18 meses
**Costo estimado:** 15,000-25,000 USD

## Ruta en EE.UU. (FDA)

### Clasificación: Clase II (510(k))

### Requisitos
| Requisito | Detalle |
|:---|:---|
| **Predicate device** | Dispositivo similar ya aprobado |
| **Substantial equivalence** | Demostrar equivalencia sustancial |
| **510(k) submission** | Expediente técnico |
| **Establishment Registration** | Registro de la empresa |
| **Device Listing** | Listado del dispositivo |

**Tiempo estimado:** 3-9 meses

## Usabilidad (IEC 62366)

### Proceso
1. **Análisis de uso** (perfil de usuario, contexto)
2. **Identificación de riesgos de uso** (uso erróneo)
3. **Especificación de interfaz de usuario**
4. **Pruebas de usabilidad** (con usuarios reales)
5. **Validación final** (ausencia de riesgos residuales)

### Entregables
- Informe de análisis de uso
- Matriz de riesgos de uso
- Protocolo de pruebas
- Informe de pruebas de usabilidad
